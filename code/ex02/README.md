### Input Folder
Two folders need to be created inside the ex02/ folder. 
- glove: to store the pre-trained glove embeddings which can be downloaded from here: https://nlp.stanford.edu/projects/glove/
- input: to store the TED talk XML file containing all TED talks of a selected language which can be downloaded from here: https://wit3.fbk.eu/mono.php?release=XML_releases&tinfo=cleanedhtml_ted
